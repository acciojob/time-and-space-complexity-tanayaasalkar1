Time and Space Complexity

Time Complexity measures how efficiently an algorithm runs as the input size increases. It is represented using Big O notation, such as O(n), where O(n) means the execution time grows linearly with the number of input elements.

Space Complexity measures the amount of memory an algorithm uses while executing. It is also expressed using Big O notation, such as O(n), where O(n) means the algorithm requires memory proportional to the input size.
